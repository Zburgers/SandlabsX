================================================================================
  üß™ API TESTING
================================================================================

Health Check:
  curl http://localhost:3001/api/health

List Nodes:
  curl http://localhost:3001/api/nodes

Create Node:
  curl -X POST http://localhost:3001/api/nodes \
    -H "Content-Type: application/json" \
    -d '{"name":"my-node"}'

Start Node:
  curl -X POST http://localhost:3001/api/nodes/<ID>/run

Stop Node:
  curl -X POST http://localhost:3001/api/nodes/<ID>/stop

Wipe Node:
  curl -X POST http://localhost:3001/api/nodes/<ID>/wipe

================================================================================
  üêõ TROUBLESHOOTING
================================================================================

Kill All Processes:
  pkill -f "next dev"
  pkill -f "node.*server"
  docker-compose down

Check Ports:
  netstat -tlnp | grep 3000      # Frontend
  netstat -tlnp | grep 3001      # Backend
  netstat -tlnp | grep 8081      # Guacamole

Clean Restart:
  ./stop-all.sh
  docker-compose down
  rm -f backend.pid frontend.pid
  rm -f backend.log frontend.log
  ./run-all.sh
